Theme Support

Dark Mode Implementation

This design system supports both automatic and manual theme switching using CSS variables and the prefers-color-scheme media query.

How it Works

  • Automatic: Respects system preference via prefers-color-scheme
  • Manual Override: Use data-theme="dark" or data-theme="light" on the <html> element
  • CSS Variables: All colors switch automatically through CSS custom properties
  • No JavaScript Required: Theme switching uses only CSS (though JS can be used for manual toggle)

Accessibility in Dark Mode

All color combinations maintain WCAG AA contrast ratios:

  • Text on background: minimum 4.5:1
  • Large text: minimum 3:1
  • Interactive elements: minimum 3:1
  • Focus indicators: clearly visible

Try navigating with the keyboard (Tab key) to see focus states in dark mode.
